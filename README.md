# Team Celebration Lunch Options

A static HTML/CSS/JS comparison page for choosing a team celebration lunch near 3333 Scott Blvd in Santa Clara, CA.

## Local Preview

Run a local static server from this folder:

```sh
python3 -m http.server 8000
```

Then open:

```text
http://127.0.0.1:8000/
```

## GitHub Pages

This site is ready to publish from the repository root on the `main` branch.

In GitHub, go to:

```text
Settings -> Pages -> Build and deployment -> Deploy from a branch
```

Use:

```text
Branch: main
Folder: / (root)
```
